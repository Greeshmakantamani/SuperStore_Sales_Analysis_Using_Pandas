📊 SuperStore Sales Analysis using Pandas


📌 Project Overview

This project analyzes SuperStore sales data using Python and Pandas.
Basic exploratory data analysis (EDA) was performed to understand sales and profit patterns.

🛠 Tools Used
Python 
Pandas
Matplotlib

📁 Dataset

The dataset contains retail sales information including:

Order details
Customer information
Product details
Sales and Profit data

🔍 Work Done
Data loading using Pandas
Data exploration
Grouping and basic analysis
Simple visualization using Matplotlib

📊 Visualization
Scatter plot using Matplotlib

📈 Key Insight
Sales and Profit data were analyzed to understand basic patterns in the dataset
🚀 How to Run This Project
Clone the repository
Install required libraries:
pip install pandas matplotlib
Open Jupyter Notebook
Run the .ipynb file
